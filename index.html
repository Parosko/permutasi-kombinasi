CALCOMA

Kalkulator Permutasi & Kombinasi

Jenis Perhitungan: Permutasi
n = 0, r = 0

Panduan Penggunaan Kalkulator

Berikut adalah penjelasan untuk tombol-tombol yang ada pada kalkulator ini:

  • Tombol Angka (0-9): Digunakan untuk memasukkan angka untuk n dan r.
  • Ubah Jenis: Mengubah jenis perhitungan antara Permutasi (P) dan Kombinasi (C).
  • Set n: Menetapkan nilai n (jumlah total elemen).
  • Set r: Menetapkan nilai r (jumlah elemen yang dipilih).
  • Hitung: Menghitung hasil berdasarkan nilai n dan r, sesuai dengan jenis perhitungan yang dipilih.
  • Edit n: Mengedit nilai n yang sudah dimasukkan.
  • Edit r: Mengedit nilai r yang sudah dimasukkan.
  • C: Membersihkan layar dan mengatur ulang nilai n dan r.
  • ←: Menghapus digit terakhir yang dimasukkan.

Anda dapat mengubah antara perhitungan Permutasi dan Kombinasi dengan menekan tombol "Ubah Jenis".
